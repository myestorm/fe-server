# fe-server

前端基础开发仓库，包含线上node服务；基础包含vue2，vue-cli3，多页应用；以及基本的proxy服务。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
